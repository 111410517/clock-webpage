# 即時時鐘網頁作業

這是一個仿照老師範例製作的即時時鐘網頁，具備現代感的深色主題與玻璃擬態 (Glassmorphism) 設計。

## 技術堆疊

- HTML5
- CSS3 (Vanilla CSS)
- JavaScript (ES6+)
- Google Fonts (Inter, Noto Sans TC)

## 功能特點

- **即時更新**：每秒更新一次時間與日期。
- **日期顯示**：包含年份、月份、日期及星期。
- **玻璃擬態**：半透明背景、模糊濾鏡、細微邊框及陰影效果。
- **互動效果**：包含卡片懸停動畫與滑鼠移動視差效果。
- **響應式佈局**：在行動裝置上會自動調整至螢幕中央。
